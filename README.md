## 🌎 English

FitBet is a web platform for creating and participating in fitness bets between friends. Challenge your friends with goals like weight loss, muscle gain, running, and other physical activities - without involving money, only health and motivation.

### 📋 Overview

FitBet allows you to:
- Create fitness challenges for your friends
- Track your progress across various physical activities
- Participate in healthy competitions to stay motivated
- Achieve health goals in a social and fun environment

### 🛠️ Technologies

This project uses the following technologies:

- **Frontend**:
  - [Next.js 14+](https://nextjs.org/)
  - [React](https://reactjs.org/)
  - [TypeScript](https://www.typescriptlang.org/)
  - [TailwindCSS](https://tailwindcss.com/) for styling
  - Animated components with CSS and JavaScript

### 🏗️ Project Structure

The project is organized as follows:

```
src/
├── app/               # Next.js configurations and routes
├── components/        # Reusable React components
│   ├── ui/            # Basic UI components
│   ├── Hero.tsx       # Home page Hero component
│   ├── Header.tsx     # Website header
│   └── ...
├── lib/               # Utility functions, hooks, etc.
└── utils/             # Helper functions
```

### 🔧 Installation and Usage

#### Prerequisites
- Node.js 18.0.0 or higher
- npm or yarn

#### Installation Steps

1. Clone the repository
```bash
git clone https://github.com/seu-usuario/fit-bet.git
cd fit-bet
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### 🚢 Deployment

To build and deploy the project:

```bash
npm run build
npm run start
```

The project is compatible with various hosting providers such as Vercel, Netlify, and others.

#### Recommended Deployment Options

1. **Vercel (Easiest)**: 
   - Connect your GitHub repository
   - Vercel will automatically detect your Next.js project

2. **Netlify**: 
   - Build command: `npm run build`
   - Publish directory: `.next`

### 🧠 Concepts

FitBet is based on some key concepts:

- **Healthy Bets**: Friendly competitions focused on health, not money
- **Progress Monitoring**: Tracking physical activities and health metrics
- **Social Motivation**: Using your social circle to maintain motivation
